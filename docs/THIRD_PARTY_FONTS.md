# Third-Party Fonts

This project uses the following third-party fonts in the desktop UI:

- Geist / Geist Mono
  - Source: <https://vercel.com/font>
  - Package: `@fontsource-variable/geist`, `@fontsource-variable/geist-mono`
  - License: OFL-1.1

Prism uses Geist / Geist Mono for Latin text and code-oriented identifiers. CJK text falls back to system UI fonts for better small-size clarity.
